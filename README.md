# Tienda Online en Consola (Java)

Proyecto integrador para repasar POO en Java.

## Requisitos cumplidos
- Enum TipoProducto y EstadoPedido
- Clase abstracta Producto + 3+ implementaciones concretas
- Interface Vendible
- Clase Pedido con lista de productos, cálculo de total con IVA, cambio de estado
- Catálogo y demostración en main

## Cómo ejecutarlo
1. Clonar el repo
2. Abrir en IntelliJ / Eclipse / VS Code
3. Ejecutar la clase principal (TiendaApp / Main)

## Tecnologías
- Java 17+ 
- Sin dependencias externas ( JDK)
